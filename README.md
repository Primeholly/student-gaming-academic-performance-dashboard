# student-gaming-academic-performance-dashboard
An interactive dashboard analysing the relationship between gaming behaviour and academic performance among students. This project explore key factor such as gaming duration, device usage, sleep patterns, class attendance, and emotional responses using data  visualisation tools to uncover actionable insight for student, educators, and policymakers.

📌 Introduction
In today’s digital age, gaming has become an integral part of student life, raising important questions about its impact on academic performance. This project presents a data-driven dashboard that explores the relationship between students’ gaming behavior and their academic outcomes.
Using a structured dataset, the analysis examines key factors such as gaming duration, device type, class attendance, sleep patterns, and emotional responses. The goal is to move beyond assumptions and provide evidence-based insights into how gaming habits influence student performance.
Ultimately, this project shows that the issue is not gaming itself, but how it is managed, promoting a balanced approach to digital engagement and academic success.
🎯 Project Objectives
Analyze the impact of gaming habits on academic performance
Identify behavioral factors influencing CGPA and student outcomes
Examine emotional responses linked to gaming dependency
Provide actionable insights for students, educators, and policymakers

🔍 Key Insights
1. Gaming Behavior
Gaming is widespread and varies by duration, device type, and game category
Mobile gaming dominates student engagement
2. Academic Performance
Academic outcomes vary more with behavioral patterns than demographics
Certain game types show different academic performance trends
3. Gender Analysis
Minimal difference between male and female students
Gaming impact is consistent across genders
4. Device Usage
Mobile gaming has the highest impact
PC gaming follows closely, while dual-device users show lower engagement
5. Class Attendance
Slight difference between regular and irregular attendance
Indicates other factors (sleep, gaming duration) play a bigger role
6. Emotional Insights
Strong negative emotions when students are unable to play
Indicates emotional dependency among frequent gamers
📈 General Observations
Gaming behavior significantly influences academic outcomes
Mobile gaming is the most dominant platform
Emotional dependency is evident among active gamers
Sleep patterns, attendance, and fatigue affect performance
Academic performance varies more with lifestyle habits than demographics
💡 Recommendations
Encourage balanced gaming habits instead of total restriction
Focus on behavioral indicators like gaming time and sleep
Integrate digital well-being programs in academic institutions
Use data-driven dashboards for better decision-making
Expand research to include mental health and lifestyle factors
